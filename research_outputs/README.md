# Research Outputs

This directory contains curated final research outputs. All files have been verified to be:

- Final clean versions (no track changes, no comments, no hidden text)
- Author-finalised submissions or clean releases
- Free of external copyrighted materials

## Directory Structure

| Subdirectory | Contents |
|-------------|----------|
| `final_manuscripts/` | Final clean DOCX and PDF manuscripts |
| `final_presentations/` | Final public presentation files |
