# Public Research Output Selection

## Selection Criteria

Files were selected for public release based on the following criteria:

1. **Author-finalised versions only**: `candidate_current_clean` or `candidate_submission_package` from version mapping
2. **Clean DOCX only**: verified free of track changes, comments, and hidden text via XML inspection
3. **PDF preferred**: PDF versions included alongside clean DOCX where a confirmed export pair exists
4. **Final presentations**: verified free of revision marks and comments
5. **Author's own work only**: no external copyrighted materials

## Exclusions

The following are **not** included in this public repository:

- All DOCX files with detected track changes (5 out of 6 candidates excluded)
- Intermediate and historical drafts
- Tracked-changes versions
- Co-author revision copies and annotated versions
- Research notes, literature notes, and reference lists
- External academic papers, book chapters, policy documents, and news materials
- Award application forms and internal management records
- All inventories, classification maps, execution logs, and full audit trails
