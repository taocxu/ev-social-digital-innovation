## Curation Principles

- Duplicate references are consolidated using DOI, canonical title, authorship, publication year, and file hash where available.
- DOI metadata must be verified against an official source; missing DOIs are never inferred or invented.
- References without a DOI are retained only when a stable official repository or publisher link is available.
- Only references with an explicit open redistribution licence, or another clear permission basis, are hosted as PDFs.
- Bibliographic entries and hosted third-party PDFs remain distinct from the repository's original research outputs.

## Copyright

All hosted PDFs retain the copyright of their original authors and publishers. No modifications have been made. The repository's MIT licence covers only the code under `control/scripts/`; it does not apply to research outputs or third-party references.
