## Adding References

To add a new reference to this directory:
1. Verify the redistribution license (CC BY or equivalent).
2. Confirm the PDF is the correct Version of Record or authorised version.
3. Add the file with a clean, stable filename following the naming convention.
4. Add attribution information to this README.
5. Use `git add -f` to stage the file (PDFs are globally gitignored).
